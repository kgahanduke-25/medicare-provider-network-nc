### **1. Data Source**

- National Provider Identifier (NPI)
- Provider state and ZIP-5
- Number of Medicare beneficiaries served (Tot_Benes)

### **2. Network Construction**
A providerâ€“provider edge is created **when two providers are located in the same ZIP code**.

**Edge weight formula:**

\[w_{ij} = \frac{1}{1 + | \text{benes}_i - \text{benes}_j |}
\]

Where:
- \( \text{benes}_i \) = Medicare beneficiaries seen by provider *i*
- Higher weight â†’ providers treat similar volumes of patients
